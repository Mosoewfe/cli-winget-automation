# Winget Helper

I got tired of typing long Winget commands every time I wanted to install or update software, so I wrote a small Python script to make the process easier.

## Features

* Search for software packages
* Install applications
* Uninstall applications
* Upgrade installed applications

## Requirements

* Windows 10 or Windows 11
* Winget installed and available in your PATH
* Python 3.x

## Usage

1. Clone this repository.
2. Double-click `run.bat`.
3. Follow the prompts in the terminal.

The script should work without administrator privileges, but running `run.bat` as Administrator is recommended because some Winget operations may require elevation.
